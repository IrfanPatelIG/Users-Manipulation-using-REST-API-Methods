
# Users Manipulation - Using REST API Methods


## Tasks

**REST API - JSON**

***GET***: /users -> Get all users --Done

***GET***: api/users/1 -> Get the user with ID 1 --Done
***GET***: api/users/'id' -> Get the user with ID 'id' --Done

***POST***: api/users -> Create a new user --Done

***PATCH***: api/users/1 -> Edit user with ID 1 --Done

***DELETE***: api/users/1 -> Delete user with ID 1 --Done

***

## Installation

1. **Clone the repository**:
   ```bash
   git clone https://github.com/IrfanPatelIG/Users-Manipulation-using-REST-API-Methods.git
   cd Users-Manipulation-using-REST-API-Methods
   ```

2. **Install dependencies**:
   ```
   npm install
   ```

***

## Running the Project

1. **Start the development server**:
   ```
   nodemon index.js
   ```
